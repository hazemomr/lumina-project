

# Lumina – Premium Beauty Salon Template

A modern, elegant, and conversion-focused website template designed for beauty salons, spas, skincare clinics, and luxury wellness businesses.

Built to deliver a premium digital experience that helps businesses attract clients, build trust, and increase bookings.


🔗 **Live Demo**

https://lumina-project-wll9.vercel.app/

---

 **Why Lumina?**

Lumina is crafted for beauty brands that want a strong, modern, and luxurious online presence.
It combines clean design, smooth interactions, and optimized performance to ensure your business looks professional from the very first click.

- Nail studios

 **Quick Start**
```bash
npm install
npm run dev
```
Then open:
```
http://localhost:5173
```

---

📦 **Production Build**
```bash
npm run build
```
The final output will be generated in the `/dist` folder.
